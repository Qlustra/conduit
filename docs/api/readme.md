# API reference

This section documents the exported `github.com/qlustra/conduit` surface directly.

It is intentionally reference-oriented:

- [Operations API](operations.md) covers package-level functions, `Context`, and defaults.
- [Layout API](layout.md) covers path handles, executable files, slots, and run options.
- [Formats API](formats.md) covers codecs, typed files, and format state methods.

For conceptual guidance, use the [usage docs](../usage/layout.md). For longer walkthroughs, use the [examples](../examples/layout.md).
