# Contributing to the project

Thanks for your interest in contributing!
This project welcomes pull requests, ideas, and discussions — but please read this short note before submitting code.

---

## Developer Certificate of Origin ([DCO](https://developercertificate.org/))

By contributing to this project, you agree to the Developer Certificate of Origin (DCO), version 1.1, and certify that:

> (a) The contribution is your original work, or  
> (b) You have the right to submit it under the same license as this project (Apache 2.0).

This ensures that all code can be distributed under the project’s existing open-source license without legal ambiguity.

You sign the DCO by **adding a “Signed-off-by” line** to your commit message, like this:

```
Signed-off-by: Your Name your.email@example.com
```

You can do this automatically using:

```bash
git commit -s
```

----

## Code Guidelines

- Keep PRs focused and concise.
- Follow existing code style and structure.
- Be descriptive in commit messages.
- Respect the project’s license.
